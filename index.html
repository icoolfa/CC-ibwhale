使用说明

🔑 快捷键与操作

  • Ctrl+C — 在输入框中复制选中内容；在终端内容页中终止/关闭当前进程
  • 输入框 — 仅用于输入命令，不参与交互式选择（如数字选项）
  • 交互式选择 — 需将鼠标移到终端内容区，按数字键或键盘方向键选择
  • Ctrl+V — 粘贴文本到终端
  • Enter — 发送输入栏内容到终端
  • 右键菜单 — 复制 / 粘贴 / 全选 / 清屏
  • 终止进程 — 点击顶部 按钮
  • 重启进程 — 点击顶部 按钮

💬 对话管理

  • 新对话 — 点击侧栏「新对话」按钮，创建独立终端
  • 重命名 — 双击对话标题，输入新名称
  • 关闭对话 — hover 对话条目，点击 × 按钮

🧠 模型切换

  • 选择模型 — 点击顶部模型选择器，切换已配置的大模型
  • 自定义模型 — 在 API 配置中输入任意模型 ID

⚙️ API 配置

  • 打开设置 — 点击左下角钥匙图标
  • 服务商 — 支持 OpenAI / Claude / DeepSeek / 阿里云 / 智谱 / 月之暗面 / OpenRouter / 本地模型
  • 数据保存 — 所有配置存储在本地文件（非浏览器 localStorage），重装不丢失
